Document APIs

这部分包含一下API

对于单个document的APIs:

  • index APIs
  • get APIs
  • delete APIs
  • update APIs

对于多个document的APIs：

  • muti get API
  • bulk API

所有的CRUD api都是对单一的index操作的，index的参数可以是一个index的名字或者指向一个index的alias

一、Index API

index api 在一个index里添加或者更新一个内容是json格式的document，使这个document是可以搜索的。

The following example inserts the JSON document into the “twitter” index, under a type called “tweet” with an id of 1:

如果index不存在，index操作会自动生成一个index，可以设置action.auto_create_index: false，禁止index自动生成。

如果mapping没有设置，也会自动生成一个mapping，可以设置index.mapper.dynamic: false，禁止mapping自动生成
